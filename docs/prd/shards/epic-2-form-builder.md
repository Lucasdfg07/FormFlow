# Epic 2: Form Builder

> **PRD Ref:** FormFlow PRD v1.0.0
> **Prioridade:** P0 (Must Have — MVP)
> **Requisitos:** FR-002, FR-003
> **Depende de:** Epic 1

---

## Escopo do Épico

Editor visual drag-and-drop para criação de formulários. Inclui todos os 17 tipos de campo, painel lateral, edição inline, preview em tempo real, lógica condicional e auto-save.

## Requisitos Cobertos

### FR-002: Editor Visual
- Drag-and-drop, painel lateral, edição inline
- Preview em tempo real, duplicar/deletar
- Lógica condicional, Welcome/Thank You screens
- Auto-save

### FR-003: Todos os 17 Tipos de Campo
- Texto (curto, longo), Choices (MC, Checkbox, Dropdown)
- Rating, NPS, Yes/No, Date, Email, Phone, URL
- File Upload, Matrix, Signature, Statement, Group

## Stories Estimadas

- Story 2.1: Estrutura do builder + drag-and-drop
- Story 2.2: Campos de texto e choices
- Story 2.3: Campos numéricos, data, contato
- Story 2.4: Campos avançados (upload, matrix, signature)
- Story 2.5: Lógica condicional
- Story 2.6: Welcome/Thank You screens + auto-save
- Story 2.7: Preview em tempo real
